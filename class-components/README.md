# React Components

This project demonstrates the basics of **React Components** using both **Functional Components** and **Class Components**. It also includes an example of converting a Class Component into a Functional Component using React Hooks.

## Topics Covered

### Functional Components
- What are Functional Components?
- Creating Functional Components
- Returning JSX
- Props in Functional Components
- Advantages of Functional Components

### Class Components
- What are Class Components?
- Creating Class Components
- `render()` Method
- `this.props`
- `this.state`
- `setState()`
- Introduction to Component Lifecycle
- Class Component vs Functional Component

### Practice Components
- Header
- Navbar
- Footer
- Student Card
- Employee Card
- Product Card

### Hooks
- Convert a Class Component into a Functional Component using the `useState` Hook

## Prerequisites

- Node.js (v18 or later recommended)
- npm (comes with Node.js)

## How to Run the Project

1. Clone the repository.

```bash
git clone https://github.com/anuj-tiwari-iphtech/Advanced-react-practice.git
```

2. Navigate to the project folder.

```bash
cd class-components
```

3. Install dependencies.

```bash
npm install
```

4. Start the development server.

For Create React App:

```bash
npm start
```

For Vite:

```bash
npm run dev
```

5. Open your browser:

- Create React App: `http://localhost:3000`
- Vite: `http://localhost:5173`

## Learning Outcomes

After completing this project, you will be able to:

- Create Functional and Class Components.
- Pass data using props.
- Manage state using `this.state` and `setState()`.
- Understand the React component lifecycle.
- Use the `useState` Hook.
- Convert Class Components into Functional Components.
