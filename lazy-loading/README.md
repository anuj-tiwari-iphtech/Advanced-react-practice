#  React Lazy Loading & Code Splitting

This project demonstrates how to improve React application performance using **Lazy Loading**, **Code Splitting**, and **React Suspense**. It includes examples of route-based and component-based lazy loading with a reusable loading spinner.

---

##  Topics Covered

- What is Lazy Loading?
- Why Lazy Loading?
- Code Splitting
- React.lazy()
- Suspense
- Fallback Loader
- Route-based Lazy Loading
- Component-based Lazy Loading
- Performance Benefits

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/anuj-tiwari-iphtech/Advanced-react-practice.git
```

### 2. Navigate to the project folder

```bash
cd lazy-loading
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

For Vite:

```bash
npm run dev
```

For Create React App:

```bash
npm start
```
